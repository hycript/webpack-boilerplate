webpack-boilerplate
===========
> a boilerplate for vue, react, other es6 project which using webpack

## Requirements
nodejs
npm
yarn

## Usage
clone git or download 

``
$ git clone https://github.com/hycript/webpack-boilerplate.git <yourAppName>
$ cd <yourAppName>
`` 

then 

``yarn``

## Development
develop with webpack-dev-server

``
npm start
``

## Deployment
``
npm run build
``
